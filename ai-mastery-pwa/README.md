# AI Product Mastery — 30-Day Program

A PWA (Progressive Web App) for the AI Product Mastery course. Install it on your phone's home screen for an app-like experience.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project → Select this repo
3. Framework: **Other** (static HTML)
4. Deploy

## Files

- `index.html` — Full course content (Weeks 1-2: Sessions 1-10 + 2 exams with model answers)
- `manifest.json` — PWA manifest
- `sw.js` — Service worker for offline support  
- `icon-192.png` / `icon-512.png` — App icons

## Install as PWA

On mobile, visit the Vercel URL → Browser menu → "Add to Home Screen"

## Updates

When new weeks are added, just replace `index.html` and push. Vercel auto-deploys.
